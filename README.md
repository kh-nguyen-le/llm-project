# **A Yu-Gi-Oh! Card game AI assistant with a simple UI**

> Built with Llama

Mechanisms are rudimentary; accuracy of model is not guaranteed

install requirements with
```
pip install -r requirements.txt
```
run with 
```
streamlit run main.py
```
